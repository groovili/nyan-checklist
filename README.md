# Nyan Checklist

Simple app for task management where you can create to do list.

![nyan-screen-safari](https://user-images.githubusercontent.com/7287535/46659898-7f041980-cbbe-11e8-9b04-e38277733a49.png)

# Stack

1. `React 16`
2. `Webpack 4`
3. `Babel 6`

# Installation

1. `npm install`
1. `npm run build`
2. `npm run start`
